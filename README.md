An alfred workflow to convert unixtime to human readable datetime format.

**Usage:**
`ut <unixtime>`

**Supported format:**
- ISO 8601                            
- ISO 8601 without 'T'                
- Locale dependent                    
- Time only (HH:MM:SS)                
- Date only                           
- UTC string                          
- Custom format (YYYY-MM-DD HH:MM:SS)
